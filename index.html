Langages les plusUtilisés

HTML

le contenu de votre page pourra être structuré en un ensemble de paragraphes, une liste à puces ou avec des images et des tableaux de données.

CSS 3

Le CSS correspond à un langage informatique permettant de mettre en forme des pages web (HTML ou XML). Ce langage est donc composé des fameuses « feuilles de style en cascade »

Java Script

C'est un langage de programmation qui permet de créer du contenu mis à jour de façon dynamique, de contrôler le contenu multimédia, d'animer des images, et tout ce à quoi on peut penser.

WordPress

C'est un outil permettant de gérer et créer simplement le contenu de votre site sans avoir à taper de code.Il permet de gagner beaucoup de temps et offre de très nombreuses fonctionnalités.

Angular

C'est un Framework Google basé sur du TypeScript qui permet de dynamiser l'expériences web de nos Clients.

Java

JAVA permet de tout coder, des applications mobiles aux logiciels d'entreprise en passant par les applications de big data et les technologies côté serveur.

Docker

De la même manière qu'une machine virtuelle virtualise le matériel serveur, les conteneurs virtualisent le système d'exploitation d'un serveur.
